Message Overview

The w2message object allows you to create message box dialogs. To use, w2message made with the global method, so you can use it with $.MsgBox.show(options) or $().w2message.show(options).
A mask can be created in any of the following ways:

Alert

or

Confirm

or

Prompt

or

Prompt Multiline

or

Complete Mode
