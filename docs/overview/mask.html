Mask Overview

w2mask plugin can mask DOM elements while their content is loading or changing to prevent user interactions and inform that some background task is still running.
A mask can be created in any of the following ways: Unmasking
To remove a previously displayed mask from an element (or multiple elements) call unmask() without any parameters: Checking if an element is masked
You can use w2isMasked() method on a single element to check if it is currently masked. Please note that this method will return false while mask is delayed.
