Enables tree node or nodes with id.
enable(id1, [id2], ...)
id string, id of the tree node
Returns integer.

Description

The node(s) must exists in the .nodes array of the tree or its nested nodes. The node is identified by the .id property. The method will set .disabled property of the node to false. It returns the number of enabled nodes that were disabled before.
