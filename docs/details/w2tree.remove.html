Removes nodes from the tree.
remove(id1, id2, ...)
id string, id of the tree node
Returns integer.

Description

Removes a tree node or nodes from the .nodes array and refreshes the tree. The tree node is identified by the .id property. it will search the entire tree including all its nested nodes. The method returns the number of nodes it removes.
