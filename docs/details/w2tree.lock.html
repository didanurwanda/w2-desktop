Locks the tree.
lock(message, [showSpinner])
message string, message
showSpinner boolean, optional, spinner indicator
Returns undefined.

Description

This method will lock the tree by added a semi-transparent div on top of its content. The user will not be able to interact with the tree or its content until the tree is unlocked with .unlock(). This method will call w2utils.lock() to perform the locking.
If you have following tree: You have 3 options how you want to lock the content.
Lock with a message and a spinner: Lock w/o a message, but with a spinner: Lock w/o a message and w/o a spinner:
