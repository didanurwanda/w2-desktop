Expands all collapsed children of specified node.
expandAll([parent])
parent string, id of the tree node
Returns undefined.

Description

The node must exists in the .nodes array of the tree or its nested nodes. The node is identified by the .id property. The method will set .expanded property of all children of the node to true. If you do not specify parent argument, then all collapsed nodes of the entire tree will be expanded.
