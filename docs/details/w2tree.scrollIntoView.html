Scroll nodes into view
scrollIntoView([id])
id string, optional, id of the tree node
Returns undefined.

Description

This method will scroll node with id = id into view. If no node specified, then selected node will be scrolled into view. The tree node is identified by the .id property. it will search the entire tree including all its nested nodes. The method returns the number of nodes it removes.
