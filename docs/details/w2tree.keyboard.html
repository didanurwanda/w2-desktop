Indicates if tree should listen to keyboard.
Boolean, default = true
If .keyboard property is set to true, the tree will add event listener to listen for keyboard events. There are several keyboard keys linked to the tree actions:
  • Arrow up - previous node
  • Arrow down - next node
  • Arrow right - expand node
  • Arrow left - collapse node
This property can be defined during object creation: Or any time after the tree has been created: Internally, the feature is implemented by w2utils.keyboard object. If there more then one tree on the screen and all of them have .keyboard set to true, then the last tree where user selects a record will active keyboard listener.
