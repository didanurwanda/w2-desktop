Selects tree node with id.
select(id)
id string, id of the tree node
Returns undefined.

Description

The node must exists in the .nodes array of the tree or its nested nodes. The node is identified by the .id property. The method will set .selected property of the node to true and will set .selected property of the tree to selected node id. Previously selected node is unselected (if any).
