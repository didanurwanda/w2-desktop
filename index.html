Smart Grid View

Any grid has a header, column, record headers, numbers, footers and select records in the column. You can show/hide then at any time. Moreover, there are several other features such as master detail, tree, inline editing, multi select, column groups, advanced search, and so forth.

Layout Bordered

Layout management with concepts like desktop applications. Make your Web application look more tidy and modern. There are different modes layouts such as Top, Bottom, Left, Right, Main, and Preview. Moreover, there are also other smart features such as resizable, toolbar, tabs, and can be integrated with other feature from W2-desktop.

Simple Tree View

TreeView provides convenience in making a vertical menu with simple and beautiful. There are some features that can support your activities or to provide a more attractive appearance such as changes icons, context menus, and so forth.

About

W2-desktop developed based library w2ui by adding some other features in order to make it more powerful library. W2-desktop you can download for free because this library uses the MIT license.
