Large Form

If you have a large form with many fields you can render the form in a variety of ways. There is no difference how you position HTML. The w2form object binds to the fields that you specify. There are several helper classes for variable width for labels and fields. In the exampble below you can see wd-span4 and wd-span5 classes, where wd-span4 = 80px and wd-span5 = 100px. There are wd-span1, ..., wd-span10, where each step is 20px.
